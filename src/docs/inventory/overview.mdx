# Inventory - Overview

The inventory app is to track, manage inventory with supplier items, supplier. 

### The user will be able to:
- Add suppliers, supplier items
- Manage onHand quantity
- Add Purchase orders
- Create bulk items, sachets items in a supplier item
- Create Work orders
- Track history and quantities- consumed, awaiting, committed
