# Create File

This method can be used to create a new file in filesystem

### API

```js
   createFile(path, content) => Promise<Result>
```

| param   | type   | description                  |
| :------ | :----- | :--------------------------- |
| path    | string | Path where to store the file |
| content | string | Content to store in the file |

### Mutation

```js
mutation createFile($path: String, $content: String) {
  createFile(path: $path, content: $content) {
    ... on Error {
      success
      error
    }
    ... on Success {
      success
      message
    }
  }
}
```

### Variables

```js
{
  "path": "appname/data/entityname/filename.json",
  "content": "hello world!"
```
