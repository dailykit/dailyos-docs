# Create Folder

This method can be used to create a new folder in filesystem

### API

```js
   createFolder(path) => Promise<Result>
```

| param | type   | description                    |
| :---- | :----- | :----------------------------- |
| path  | string | Path where to store the folder |

### Mutation

```js
mutation createFolder($path: String) {
  createFolder(path: $path) {
    ... on Error {
      success
      error
    }
    ... on Success {
      success
      message
    }
  }
}
```

### Variables

```js
{
  "path": "appname/data/entityname/foldername"
}
```
