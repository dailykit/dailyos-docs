# Introduction

DailyOS Recipe app allows you to create and manage recipes with ease.

## Some Salient Features
 - Add Recipe Information for multiple servings
 - Add Information related to cuisine, chef, utensils

Recipes are the soul of any food business and operations. 
The art of writing recipes have been practiced from ages and this time is no different.
