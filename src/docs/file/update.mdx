# Update File

This method can be used to update a file in the filesystem.

### API

```js
    updateFile(path, content) => Promise<string>
```

| param   | type   | description                  |
| :------ | :----- | :--------------------------- |
| path    | string | Path where to save your file |
| content | string | Content to store in the file |
